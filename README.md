## Description
TSD OSINT is a Python-based OSINT (Open Source Intelligence) tool designed to gather information from social media accounts, emails, phone numbers, websites, and more. It’s lightweight, easy to use, and ideal for security researchers, penetration testers, and OSINT enthusiasts.

---

## Features
- Search social media profiles by username
- Email and phone number lookup
- Extract public data from websites
- Multiple OSINT modes for detailed information
- Red colored, animated terminal interface for better user experience

## Installation
Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/TSD-OSINT.git
cd TSD-OSINT 

## Install dependencies

pip install -r requirements.txt

(Optional) Run auto-installer for Linux:

bash install.sh

## Run the tool

python tsd_osint.py

Follow the on-screen menu to select the desired OSINT mode.
